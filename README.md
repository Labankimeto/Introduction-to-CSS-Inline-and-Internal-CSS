Requirements
Part A: Internal CSS
Create an internal stylesheet inside the <head> section of the HTML document using the <style> tag.
Use internal CSS to style the following:
•	The body
•	The header
•	All h2 headings
•	All article elements
•	All paragraph (p) elements
•	The .gray class
•	The #second ID
•	The .highlight class
Part B: Inline CSS
Use the style attribute directly within HTML elements to style:
•	One <h1> element
•	One <h2> element
•	One <article> element
•	One paragraph (<p>) element
•	One <span> element
Use different CSS properties for each example.
Part C: Creativity
Enhance the overall appearance of the webpage by incorporating additional styling techniques, such as:
•	Rounded corners
•	Box shadows
•	Letter spacing
•	Text transformations
•	Hover effects
•	Appropriate colour combinations
